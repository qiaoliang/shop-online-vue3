# 集成点和外部依赖

### 外部服务

| 服务     | 目的       | 集成类型 | 关键文件          |
| -------- | ---------- | -------- | ----------------- |
| 后端 API | 数据交互   | REST API | `src/apis/`       |
| Vant UI  | 移动端组件 | 组件库   | `src/components/` |
| VueUse   | 组合式工具 | 工具库   | `src/hooks/`      |

### 内部集成点

- **前端路由**：Vue Router 管理页面跳转，需要特定路由守卫
- **状态管理**：Pinia 管理全局状态，模块化组织
- **API 调用**：Axios 封装 HTTP 请求，统一错误处理
